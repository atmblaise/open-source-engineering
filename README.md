Open source engineering software, technical contributions, and reusable computational tools for process engineering.

## Purpose

The Open Source Engineering workspace documents contributions to engineering software, the development of reusable computational tools, and participation in the open source engineering community.

The objective is to improve engineering practice by developing, maintaining, and sharing software, documentation, and technical resources that are openly available to engineers, researchers, and students.
